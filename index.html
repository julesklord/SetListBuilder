FMG Setlist Builder

A tool for musicians to build, organize, and export setlists — with AI-powered song lookup and mood-based suggestions. All data stays in your browser. No account required.

Getting started

1

Select the instruments available tonight in the sidebar. Choose any combination of Guitar, Piano, and Winds — the generator will prioritize songs suited to your selection.

2

Choose the number of sets (2, 3, or 4) and duration per set (~40, ~45, ~55, or ~60 min).

3

Select which genres to include in the generation. Choose from Blues, Rock, Pop, Soul, Funk, R&B, Ballad, Reggae, Latin, Jazz, or Country. Check the boxes next to the genres you want.

4

Click Generate Setlist. The algorithm builds an energy curve for each set — opening low, building through the middle, peaking in the final set. Songs are selected only from your chosen genres and instruments.

5

Drag and drop songs within a set to reorder them. Use the × button to remove a song, or the dropdown at the bottom of each set to add one.

6

Give your setlist a name in the title field and click Export to download as HTML, JSON, or copy as text.

Instruments & generation

The setlist generator intelligently prioritizes songs based on your selected instruments and genres.

How instruments work

When you select Guitar, Piano, or Winds in the sidebar, the algorithm prefers songs tagged for those instruments. Each song has instrument tags (G, P, V) indicating how well it suits each instrument. If there aren't enough songs matching your selections, the algorithm falls back to the full pool.

Energy curve

Each setlist follows a musical energy pattern:

  • Set 1 & 2: Opens slow/mellow, builds toward mid-energy
  • Final set: Climbs higher, ends on a peak

This keeps audiences engaged and provides natural peaks and valleys in your performance.

AI features

Important: AI features require an API key from one of three providers: Claude (Anthropic), Gemini (Google), or ChatGPT (OpenAI). Your key is stored only in your browser's localStorage — it is never sent to any server other than the selected API provider directly from your browser. Get keys from: Anthropic | Google | OpenAI. API usage is billed per request (very low cost).

Choosing an API provider

Use the dropdown in the settings bar to select which AI provider to use. Each provider has similar performance for music metadata tasks. Claude and ChatGPT tend to be slightly more reliable for complex requests, while Gemini is often faster and cheaper.

Song lookup

Type a song name and artist (e.g. "Red House – Jimi Hendrix") and the AI will automatically fill in the genre, key, BPM, chord progression, energy level, and best instrument fit. You can select from any of the 11 supported genres. Review and click Add to pool to save it.

Suggest by mood

Describe what you're looking for (e.g. "upbeat funk with bass, energetic, Dm") and the AI will suggest 5 songs that fit — with full metadata ready to add to your pool. Works across all genres.

Song pool

The pool is your master library of songs. It comes pre-loaded with 90 carefully curated songs spanning 11 genres: Blues, Rock, Pop, Soul, Funk, R&B, Ballad, Reggae, Latin, Jazz, and Country — ensuring you have plenty of options without needing AI to generate setlists. You can:

  • Add songs manually via + Manual
  • Add songs via AI lookup or suggestion
  • Edit any song's metadata (genre, key, BPM, progression, instruments)
  • Delete songs from the pool
  • Export the pool as JSON and import it on another device

Each song has an instrument tag (G = Guitar, P = Piano, V = Winds) that controls which songs get prioritized when you select available instruments and genres for generation.

Export & share

All setlists and song pool data live in your browser's localStorage. To move your data to another device or share with collaborators, use the export features:

  • Download HTML — a self-contained styled HTML file of your setlist, ready to print or distribute
  • Copy as text — plain text version perfect for messaging, emails, or notes
  • Export JSON — raw setlist data for archiving or detailed analysis
  • Export pool JSON (in Pool view) — save your entire song library as a file
  • Import pool JSON (in Pool view) — load a previously exported pool (merges with existing songs, no duplicates)

Deploying to GitHub Pages

Deploy this app publicly so your band or team can access it from anywhere.

1

Create a GitHub account at github.com if needed.

2

Create a new public repository (e.g. fmg-setlist).

3

Rename this file to index.html and commit it to main.

4

Go to Settings → Pages → Source → main branch. Save.

5

In a few minutes, your app will be live at yourusername.github.io/fmg-setlist.

Important: Each visitor has their own independent song pool and setlist data stored locally in their browser. Data doesn't sync between users — use JSON export/import to share pools across devices.

Tips & tricks

  • Genre mixing: Try selecting 2-3 genres for variety. Blues + Soul creates a cohesive vibe, while Pop + Rock + Soul broadens appeal.
  • Song sequencing: Drag rows to manually fine-tune the energy curve. Avoid two high-energy songs back-to-back.
  • Test the algorithm: Generate several setlists by unchecking/checking genres. The algorithm uses randomization, so each run is slightly different.
  • Add custom songs: Don't see a song you want? Go to Pool and add it manually or use AI lookup.
  • Backup your pool: Export your song pool as JSON regularly. Store it somewhere safe in case your browser clears localStorage.
  • Share setlists: Export as HTML for a professional-looking document to share with band members or send to venues.
